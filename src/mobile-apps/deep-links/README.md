# Deep Links

Deep links allow the user to open the app from a link (instead of opening the link in the browser)

Navigation is handled by `DeepLinksListener.tsx`

Followed (among other guides) https://capacitorjs.com/docs/guides/deep-links

`to-copy-in-well-known` folder is a copy of the `.well-known` folder in https://areaprivada.ufd.es/.well-known/
